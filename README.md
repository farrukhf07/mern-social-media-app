# MERN Social Media Application

A full-stack social media application built using the MERN stack (MongoDB, Express.js, React, Node.js). The backend is developed with Node.js and Express, featuring RESTful APIs secured with JWT authentication and tested using Postman. MongoDB is used for efficient data storage and management.

The frontend is built with React and seamlessly integrated with the backend APIs to provide a smooth and responsive user experience. The application supports user authentication, authorization, and core social media functionalities.

# Tech Stack:

MongoDB

Express.js

React.js

Node.js

JWT Authentication

REST APIs

Postman (API Testing)
